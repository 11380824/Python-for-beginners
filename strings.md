### Strings in  Python
* Strings in python can be in either double quotes or single quotes
```Python
my_str = 'Hello World'
string = "Hello"
```

#### String Concatenation
* We Concatenate strings using **+** operator 
```Python
str1 = 'Hello'
str2 = 'World'
print(str1+str2)
#This will print out Hello World
```
